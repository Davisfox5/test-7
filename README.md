# Pacheco Carpentry — pitch website

A standalone single-page static website for **Pacheco Carpentry Services**
(Houston, TX). Vanilla HTML / CSS / JS, no build step, opens in any browser.

```bash
python3 -m http.server 8000 --directory site
# http://localhost:8000
```

See [`site/README.md`](site/README.md) for editing notes, the photo-fallback
behaviour, the Formspree swap, and Netlify / GitHub Pages deployment.
